## ✨ 학습 내용 요약

### HTML
- HTML 기본 구조와 문법
- 시맨틱 태그 사용법 (`<header>`, `<section>`, `<article>`, `<footer>` 등)
- 텍스트, 이미지, 하이퍼링크, 리스트, 테이블 작성
- 폼 요소 및 사용자 입력 처리

### CSS
- 선택자와 속성 이해
- 색상, 폰트, 배경 스타일링
- 박스 모델(Box Model) 이해
- Flexbox 레이아웃 기초
- 간단한 반응형 웹페이지 구성
